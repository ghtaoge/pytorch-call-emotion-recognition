# 模型限制

默认模型为 [`xmj2002/hubert-base-ch-speech-emotion-recognition`](https://huggingface.co/xmj2002/hubert-base-ch-speech-emotion-recognition)，模型卡声明 Apache-2.0 许可证。它基于 Chinese HuBERT，并使用 CASIA 中文情绪语音训练。模型卡说明所用子集约 1200 条表演录音，因此测试集成绩不能代表真实电话场景。

原始标签为愤怒、恐惧、开心、中性、悲伤、惊讶。页面只展示愤怒、开心、中性、悲伤四类：

```text
目标概率 = 原始目标概率 / 四个目标概率之和
排除概率 = 恐惧概率 + 惊讶概率
```

排除概率超过 35%、最高概率低于 45% 或前两名差值小于 12% 时，结果标记为低可靠性。该标记只说明模型输出不稳定，不是统计置信区间。

已知限制包括方言、儿童与老年声音、哭声、多人重叠、强噪声、音乐、变声、跨语言、电话压缩和非表演式真实情绪。模型不理解说话内容，也不能区分通话双方。
