# 隐私说明

- 页面和 API 默认只监听 `127.0.0.1`。
- 音频不会发送到远程推理服务。
- 项目不使用账号、Cookie、数据库、遥测或第三方分析脚本。
- 压缩音频仅在解码期间写入随机临时文件，并在 `finally` 中删除。
- 服务日志不记录原始文件名、音频内容、预测概率或临时路径。
- 模型首次下载会连接 Hugging Face；缓存完成后，推理可离线运行。

使用者仍需确保自己有权处理通话录音，并遵守所在地关于告知、同意和数据保护的要求。

### URL 音频

- URL 音频下载到内存临时缓冲区，经 FFmpeg 解码后立即用于分析
- 分析完成后，内存中的音频数据随请求结束自动释放，不在本地持久存储
- 请求日志仅记录 URL 的协议和域名部分，不记录完整路径，防止泄露内网地址
- 前端不缓存 URL，页面关闭或刷新后 URL 输入自动清除
